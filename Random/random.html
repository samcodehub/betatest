What is Random module in Python?

The random module in Python is a built-in module that provides functions for generating random numbers and other random operations. It is a part of the standard Python library and can be imported easily in any Python program.

The random module provides several functions that generate random numbers or make random selections from a sequence. Some of the commonly used functions in the random module include:

  • random(): This function generates a random float number between 0 and 1.
  • randint(a, b): This function generates a random integer between a and b (inclusive).
  • choice(seq): This function returns a random element from a sequence.
  • shuffle(seq): This function shuffles the elements of a sequence randomly.
  • sample(population, k): This function returns a random sample of k elements from a population sequence.

Using the random module, you can create programs that require randomness or chance, such as simulations, games, or data encryption. However, it is important to note that the random module generates pseudorandom numbers, which are not truly random but are generated using a deterministic algorithm.

How to use random module in Python?

To use the random module in Python, you first need to import it in your program using the import statement:

import random

After importing the random module, you can use its various functions to generate random numbers or make random selections. Here are some examples:

  1. Generate a random integer between 1 and 10:
    2. import random

random_number = random.randint(1, 10)
print(random_number)
  2. Generate a random floating-point number between 0 and 1:
    3. import random

random_float = random.random()
print(random_float)
  3. Select a random element from a list:
    4. import random

my_list = [1, 2, 3, 4, 5]
random_element = random.choice(my_list)
print(random_element)
  4. Shuffle the elements of a list randomly:
    5. import random

my_list = [1, 2, 3, 4, 5]
random.shuffle(my_list)
print(my_list)
  5. Get a random sample of 3 elements from a list:
    6. import random

my_list = [1, 2, 3, 4, 5]
random_sample = random.sample(my_list, 3)
print(random_sample)

These are just a few examples of how you can use the random module in Python. There are many other functions and use cases for generating random numbers and making random selections, so make sure to check the documentation for more information.
